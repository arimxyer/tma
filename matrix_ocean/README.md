# Matrix Ocean

Ocean waves rendered with Matrix-style code characters.

![Preview](preview.gif)

## Run

```bash
python matrix_ocean.py
# or
uv run matrix_ocean.py
```

## Features

- Wave interference patterns
- Matrix-style character palette (binary, operators, symbols)
- Green phosphor terminal aesthetic
- Dynamic color gradients
- 30 FPS animation

## Requirements

- Python 3.12+
- numpy
