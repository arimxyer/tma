# Cube

Rotating 3D cube with sub-pixel rendering for smooth visuals.

![Preview](preview.gif)

## Run

```bash
python cube.py
# or
uv run cube.py
```

## Features

- True 3D rotation on X, Y, Z axes
- Sub-pixel rendering using Unicode half-blocks
- Perspective projection with minimal distortion
- Depth-based face coloring
- Z-buffer hidden surface removal
- 60 FPS smooth animation

## Requirements

- Python 3.12+
- numpy
