# Ocean Waves

Animated ocean waves simulation using sine wave interference patterns.

![Preview](preview.gif)

## Run

```bash
python ocean_waves.py
# or
uv run ocean_waves.py
```

## Features

- Multi-layer wave interference patterns
- Dynamic blue-to-cyan color gradients
- ASCII character depth mapping
- Terminal-based ANSI rendering
- 30 FPS animation

## Requirements

- Python 3.12+
- numpy
