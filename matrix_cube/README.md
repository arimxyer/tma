# Matrix Cube

Rotating 3D cube with Matrix code rain aesthetic.

![Preview](preview.gif)

## Run

```bash
python matrix_cube.py
# or
uv run matrix_cube.py
```

## Features

- 3D rotating cube geometry
- Matrix green color scheme
- Sub-pixel rendering using Unicode half-blocks
- Perspective projection
- 60 FPS smooth animation

## Requirements

- Python 3.12+
- numpy
