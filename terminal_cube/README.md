# Terminal Cube

Retro terminal-styled rotating cube with hexadecimal glyphs.

![Preview](preview.gif)

## Run

```bash
python terminal_cube.py
# or
uv run terminal_cube.py
```

## Features

- 3D rotating cube geometry
- Retro terminal aesthetic
- Hexadecimal and tech glyph character set
- Zoomed out "data visualization" style
- 30 FPS for authentic retro feel

## Requirements

- Python 3.12+
- numpy
