# Hyper Cube

4D tesseract (hypercube) projected into 3D space with smooth rotation.

![Preview](preview.gif)

## Run

```bash
python hyper_cube.py
# or
uv run hyper_cube.py
```

## Features

- True 4D hypercube geometry (16 vertices)
- Rotation in 4-dimensional space
- Projection from 4D to 3D to 2D
- Sub-pixel rendering using Unicode half-blocks
- Perspective projection
- 60 FPS smooth animation

## Requirements

- Python 3.12+
- numpy
