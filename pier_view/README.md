# Pier View

Advanced ocean visualization with realistic wave physics and atmospheric perspective.

![Preview](preview.gif)

## Run

```bash
python pier_view.py
# or
uv run pier_view.py
```

## Features

- Wave set simulation (groups of larger waves)
- Rare rogue wave events
- Slow swell envelope breathing
- Atmospheric perspective with horizon haze
- Foam detection on wave peaks
- 60 FPS smooth animation

## Requirements

- Python 3.12+
- numpy
