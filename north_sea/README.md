# North Sea

Rough ocean view from a boat with realistic vessel motion simulation.

![Preview](preview.gif)

## Run

```bash
python north_sea.py
# or
uv run north_sea.py
```

## Features

- 6 degrees of freedom boat motion (roll, pitch, heave, yaw)
- Steep, chaotic North Sea wave patterns
- Cross-swell wave interference
- Directional wave face rendering
- Spray/foam effects on wave peaks
- 60 FPS animation

## Requirements

- Python 3.12+
- numpy
